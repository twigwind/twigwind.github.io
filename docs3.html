to use these write the color name with the color- prefix

<p class="color-red">This is Red text.</p>
<p class="color-pink">This is Pink text.</p>
<p class="color-purple">This is Purple text.</p>
<p class="color-blue">This is Blue text.</p>
<p class="color-green">This is Green text.</p>
<p class="color-yellow">This is Yellow text.</p>

These are the predefined color's but Twigwind support ✨Arbratary colors✨ like

<p class="color-rgb(252, 24, 8)">this is red<p>
<p class="color-#ffffff">this is white<p>
<p class="color-rgba(252, 0, 130, 0.39)">this is transparent pink<p>

background

to style the background use bg-

<p class="bg-rgb(252, 24, 8)">this is red<p>
<p class="bg-#ffffff">this is white<p>
<p class="bg-rgba(252, 0, 130, 0.39)">this is transparent pink<p>
