Google Fonts & Icons

Twigwind comes with pre-configured Google Fonts and Material Symbols for beautiful typography and iconography.

üí° Tip: All fonts and icons are automatically imported via css.css - no additional setup required!

Google Fonts

Twigwind includes several carefully selected Google Fonts for different use cases:

Available Font Families

  • font-serif - Classic serif fonts (Georgia, Times New Roman)
  • font-roboto - Modern sans-serif (Roboto)
  • font-fira - Monospace coding font (Fira Code)
  • font-jetbrains - Developer-friendly monospace (JetBrains Mono)
  • font-roboto-mono - Monospace variant of Roboto
  • font-sofia - Elegant cursive font (Sofia)

Material Symbols

Twigwind includes Google's Material Symbols Outlined for consistent iconography across your application.

üí° Tip: Material Symbols are vector-based and scale perfectly at any size!

Using Material Symbols

Use the material-symbols-outlined class with the icon name:

Basic Icon Usage

<span class="material-symbols-outlined">home</span>
<span class="material-symbols-outlined">search</span>
<span class="material-symbols-outlined">favorite</span>
<span class="material-symbols-outlined">settings</span>
<span class="material-symbols-outlined">account_circle</span>

Icon Configuration

Material Symbols come pre-configured with optimal settings:

  • FILL: 0 (outlined style)
  • Weight: 400 (normal weight)
  • Grade: 0 (standard grade)
  • Optical Size: 24 (default size)
  • Size: 1.5rem (24px equivalent)
  • Alignment: Middle vertical alignment
