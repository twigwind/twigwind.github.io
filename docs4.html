Size

Twigwind supports muiltiple predefined sizes which are

  • sm
  • md
  • lg
  • xl
  • 2xl

example

<div class="bg-teal size-sm mr-10 p-5">SM</div>
<div class="bg-teal size-md mr-10 p-5">MD</div>
<div class="bg-teal size-lg mr-10 p-5">LG</div>
<div class="bg-teal size-xl p-5">XL</div>

You can also use custom sizes by applying arbitrary values, for example:

<div class="bg-teal size-[10px] mr-10 p-5">Custom Size</div>
