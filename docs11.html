Animation Utilities

Twigwind provides a comprehensive set of animation utilities using a dynamic syntax pattern. All keyframes are defined in css.css and work seamlessly with the framework's JavaScript-powered utility generation.

üí° Tip: All animations work seamlessly with hover states and responsive prefixes for maximum flexibility.

Animation Syntax

Twigwind uses a dynamic animation syntax pattern:

Syntax Pattern

animate-(keyframe)-(duration)-(direction)
  • keyframe: The animation name (e.g., spin, bounce, fadeIn)
  • duration: Time with unit (e.g., 500ms, 2s, 1.5s)
  • direction: Animation direction (infinite, normal, reverse, alternate, alternate-reverse)

Available Keyframes

All keyframes are defined in css.css and include:

  • spin - 360¬∞ rotation
  • pulse - Opacity and scale pulsing
  • bounce - Vertical bouncing with easing
  • ping - Scale and fade effect
  • fadeIn - Fade in from transparent
  • fadeOut - Fade out to transparent
  • slideUp - Slide in from bottom
  • slideDown - Slide in from top
  • slideLeft - Slide in from right
  • slideRight - Slide in from left
  • zoomIn - Scale in from zero
  • zoomOut - Scale out to zero
  • shake - Horizontal oscillation
  • wiggle - Rotational oscillation
  • heartbeat - Scale pulsing
  • flash - Opacity flashing
  • rubberBand - Elastic deformation

Animation Directions

Control how animations play with different direction values:

  • normal - Play animation once in normal direction
  • reverse - Play animation once in reverse direction
  • infinite - Loop animation infinitely
  • alternate - Alternate between normal and reverse
  • alternate-reverse - Alternate starting with reverse

Example

<div class="animate-spin-2s-normal">Spin once</div>
<div class="animate-spin-2s-reverse">Spin reverse</div>
<div class="animate-spin-1s-infinite">Spin forever</div>
<div class="animate-bounce-1s-alternate">Bounce back and forth</div>
