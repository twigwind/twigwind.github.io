Animation Utilities

Twigwind provides a comprehensive set of animation utilities with predefined keyframes for creating engaging and dynamic user interfaces. All animations are defined in css.css and work seamlessly with the framework.

üí° Tip: All animations work seamlessly with hover states and responsive prefixes for maximum flexibility.

Basic Animations

Essential animations for common UI interactions, available as CSS classes:

  • animate-spin - Continuous rotation (perfect for loading spinners)
  • animate-pulse - Gentle opacity fade in/out
  • animate-bounce - Bouncing motion
  • animate-ping - Scale and fade effect (great for notifications)

Example

<div class="animate-spin bg-blue p-10">Loading...</div>
<div class="animate-pulse bg-green p-10">Pulsing element</div>
<div class="animate-bounce bg-red p-10">Bouncing box</div>
<div class="animate-ping bg-yellow p-10">Notification</div>

Entrance Animations

Smooth entrance effects for revealing content:

  • animate-fadeIn - Fade in from transparent to opaque
  • animate-slideUp - Slide in from bottom
  • animate-slideDown - Slide in from top
  • animate-slideLeft - Slide in from right
  • animate-slideRight - Slide in from left
  • animate-zoomIn - Scale in from zero

Example

<div class="animate-fadeIn bg-purple p-10">Fading in</div>
<div class="animate-slideUp bg-teal p-10">Sliding up</div>
<div class="animate-zoomIn bg-orange p-10">Zooming in</div>

Exit Animations

Smooth exit effects for hiding content:

  • animate-fadeOut - Fade out from opaque to transparent
  • animate-zoomOut - Scale out to zero

Example

<div class="animate-fadeOut bg-red p-10">Fading out</div>
<div class="animate-zoomOut bg-blue p-10">Zooming out</div>

Special Effects

Eye-catching animations for emphasis and interaction:

  • animate-shake - Horizontal shaking motion
  • animate-wiggle - Gentle rotation wiggle
  • animate-heartbeat - Rhythmic scaling (like a heartbeat)
  • animate-flash - Quick opacity flashing
  • animate-rubberBand - Elastic stretching effect

Example

<div class="animate-shake bg-red p-10">Shake on error</div>
<div class="animate-heartbeat bg-pink p-10">Like button</div>
<div class="animate-rubberBand bg-green p-10">Elastic effect</div>

Animation Control Classes

Fine-tune animation behavior with control utilities defined in css.css:

Play State

  • animate-paused - Pause the animation
  • animate-running - Resume the animation

Iteration Count

  • animate-infinite - Loop animation infinitely
  • animate-once - Play animation only once
  • animate-twice - Play animation twice

Duration

  • animate-fast - Faster animation (0.3s)
  • animate-slow - Slower animation (2s)
  • animate-slower - Much slower animation (3s)

Delay

  • animate-delay-100 - 0.1s delay
  • animate-delay-200 - 0.2s delay
  • animate-delay-300 - 0.3s delay
  • animate-delay-500 - 0.5s delay
  • animate-delay-1000 - 1s delay

Example

<div class="animate-spin animate-fast">Fast spinning</div>
<div class="animate-bounce animate-slow animate-twice">Slow bounce twice</div>
<div class="animate-fadeIn animate-delay-500">Delayed fade in</div>

<!-- Interactive control -->
<div id="controllable" class="animate-spin">Controllable</div>
<button onclick="document.getElementById('controllable').classList.add('animate-paused')">
  Pause
</button>

Dynamic Animation (JavaScript)

For advanced use cases, Twigwind also supports dynamic animation generation through JavaScript with the pattern:

animate-{type}-{duration}{unit}-{iteration}

Example

<!-- These would be processed by the JavaScript engine -->
<div class="animate-bounce-500ms-infinite">Custom bounce</div>
<div class="animate-spin-2s-normal">Custom spin</div>

Hover Animations

Trigger animations on hover for interactive effects:

Example

<div class="hover:animate-spin bg-blue p-10">Hover to spin</div>
<div class="hover:animate-bounce bg-green p-10">Hover to bounce</div>
<div class="hover:animate-shake bg-red p-10">Hover to shake</div>
<div class="hover:animate-pulse bg-purple p-10">Hover to pulse</div>

Responsive Animations

Control animations at different screen sizes:

Example

<div class="animate-pulse sm:animate-spin md:animate-bounce lg:animate-shake">
  Different animations at different screen sizes
</div>

Combining with Other Utilities

Mix animations with other Twigwind utilities for complex effects:

Example

<!-- Animation with position change -->
<div class="absolute top-50 left-50 animate-bounce 
           hover:top-20 hover:left-20 
           transition:all-ease-300">
  Bouncing element that moves on hover
</div>

<!-- Animation with color change -->
<div class="animate-pulse bg-blue hover:bg-red 
           transition:background-color-300">
  Pulsing element with color transition
</div>
