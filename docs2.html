Installation

to install twigwind vist https://github.com/helloadhavan/twigwind or run

git clone https://github.com/helloadhavan/twigwind.git

by running the command above you'll get the following files

  • version.txt: the version of twigwind

  • css.js: the real js sources file

  • css.css: the optional css file for icons,fonts, etc...

  • hljs_styles: the css folder for hljs highlighting stylesheets

  • README.md: the md file

  • hljs.js: the js file for including the hljs extention

Usage

<script src="/path/to/twigwind/css.js"></script>
<link href="/path/to/twigwind/css.css" rel="stylesheet">
⚠️ Note: Twigwind is still in development. Expect frequent updates.
