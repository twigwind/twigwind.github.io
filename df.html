Building vs Runtime Mode

Twigwind operates in two modes: Building mode and Runtime mode. Each mode serves a specific purpose and has its own advantages and use cases.

Building Mode

In Building mode, Twigwind processes your HTML and CSS files at build time. It generates the necessary CSS based on the classes used in your HTML and saves it into a css file.

Setup

to setup Building mode, install the Twigwind CLI and run the build command.

npm install -g twigwind
twigwind build --input <input-directory> --output <output-directory>
Note: Building mode is only support by versions 3.1.4 and above.

Runtime Mode

In Runtime mode, Twigwind processes your HTML and CSS files in the browser at runtime. It generates the necessary CSS on the fly based on the classes used in your HTML.

Setup

to setup Runtime mode, simply include the Twigwind JavaScript file in your HTML.

<script src="/twigwind.js"></script>
Note: Runtime mode is supported by all versions of Twigwind.
