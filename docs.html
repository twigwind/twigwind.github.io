Getting Started

Welcome to the Twigwind CSS Framework documentation. Twigwind is a dark-first utility CSS framework built with JavaScript for rapid prototyping and flexibility.

💡 Tip: You don’t need a build step — just include css.css and css.js in your HTML.

Installation

Add Twigwind to your HTML:

<link rel="stylesheet" href="css.css">
<script src="css.js"></script>
✅ Twigwind is now active — you can use its classes directly in your HTML.

Usage

Here’s a quick example:

<h1 class="color-blue size-xl mb-20">Hello Twigwind</h1>
<div class="flex:row-center-center bg-green p-10">
  <div class="bg-purple p-5 mr-10">Item 1</div>
  <div class="bg-red p-5">Item 2</div>
</div>

Utilities

Twigwind includes utility classes for:

  • Colors — e.g. bg-blue, color-red
  • Spacing — e.g. m-20, p-10
  • Flex Layouts — e.g. flex:row-center-center
  • Text styles — e.g. italic, bold, font-fira
  • Hover states — e.g. hover:bg-purple

Examples

A section with hover effects:

<div class="flex:row-left-center">
  <div class="bg-cyan hover:bg-purple size-md mr-10 p-10">Hover Me</div>
  <div class="bg-lightBlue hover:bg-amber size-md p-10">Hover Me</div>
</div>
