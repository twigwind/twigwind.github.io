Getting Started

Welcome to the Twigwind CSS Framework documentation. Twigwind is a utility-first CSS framework built with JavaScript for rapid prototyping and flexibility.

üí° Tip: Use Twigwind with runtime CSS generation or build-time optimization.

Installation

NPM Package

npm install twigwind

CDN package

Add Twigwind to your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/helloadhavan/twigwind/src/css.css">
<script src="https://cdn.jsdelivr.net/gh/helloadhavan/twigwind/src/css.js" type="module"></script>

Initialize Twigwind after DOM content loads:

<script>
document.addEventListener("DOMContentLoaded", () => {
  document.querySelectorAll("[class]").forEach(el => Twigwind.twApply(el));
  Twigwind.twInject();
});
</script>
‚úÖ Twigwind is now active ‚Äî you can use its classes directly in your HTML.

Build System (Optional)

For optimized CSS generation:

node build.js

This will generate optimized CSS files in the dist/ directory.

Utilities

Twigwind includes utility classes for:

  • Colors ‚Äî e.g. bg-blue, color-red
  • Spacing ‚Äî e.g. m-20, p-10
  • Flex Layouts ‚Äî e.g. flex:row-center-center
  • Position ‚Äî e.g. absolute, top-10, z-20
  • Animations ‚Äî e.g. animate-spin, animate-bounce
  • Gradients ‚Äî e.g. gradient-to-r-red-blue, gradient-45deg-purple-pink
  • Backgrounds ‚Äî e.g. clip-border, image-url-[URL]
  • Hover states ‚Äî e.g. hover:bg-purple
